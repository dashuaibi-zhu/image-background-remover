# 背景消除工具 MVP 需求文档

## 1. 产品概述

| 项目 | 说明 |
|------|------|
| **产品名称** | Remove BG（背景消除工具） |
| **版本** | MVP v1.0 |
| **部署平台** | Cloudflare Workers |
| **核心技术** | remove.bg API |
| **数据存储** | 无（纯内存处理） |

一句话描述：**用户上传图片，一键去除背景，下载透明底图。**

---

## 2. 目标用户

| 用户类型 | 场景 |
|----------|------|
| 电商卖家 | 商品图去背景，用于主图/详情页 |
| 设计师 | 快速抠图，放入设计稿 |
| 普通用户 | 社交头像、趣味图片处理 |

---

## 3. 功能需求

### 3.1 核心功能（Must Have）

| 编号 | 功能 | 说明 |
|------|------|------|
| F1 | 图片上传 | 支持点击上传 / 拖拽上传 |
| F2 | 格式校验 | 仅允许 JPG / PNG / WebP，最大 10MB |
| F3 | 背景消除 | 调用 remove.bg API 处理图片 |
| F4 | 结果预览 | 左右对比：原图 vs 去背结果 |
| F5 | 结果下载 | 一键下载 PNG 透明底图 |
| F6 | 重新上传 | 处理完成后可重新开始 |

### 3.2 交互体验

| 编号 | 功能 | 说明 |
|------|------|------|
| F7 | 拖拽高亮 | 拖拽文件到上传区域时视觉反馈 |
| F8 | 加载动画 | 处理中显示 spinner + 提示文字 |
| F9 | 错误提示 | 格式错误、大小超限、API 异常等友好提示 |
| F10 | 透明背景网格 | 结果图使用棋盘格背景展示透明区域 |

### 3.3 MVP 不做（Backlog）

- ❌ 批量处理
- ❌ 用户登录 / 历史记录
- ❌ 自定义背景替换
- ❌ 图片编辑（裁剪、调色）
- ❌ 付费 / 配额管理

## 4. 技术方案

### 4.1 架构

```
用户浏览器
    │
    ▼
┌─────────────────────────┐
│  Cloudflare Worker      │
│  ┌───────────────────┐  │
│  │ 前端页面（内嵌）   │  │
│  └───────────────────┘  │
│         │               │
│    /api/remove-bg       │
│         │               │
│         ▼               │
│  ┌───────────────────┐  │
│  │ 图片处理（内存）   │  │
│  │ ArrayBuffer 流转   │  │
│  └───────────────────┘  │
└─────────────────────────┘
    │
    ▼
┌─────────────────────────┐
│  remove.bg API          │
│  POST /v1.0/removebg    │
└─────────────────────────┘
```

### 4.2 关键技术决策

| 决策项 | 方案 | 理由 |
|--------|------|------|
| 部署平台 | Cloudflare Workers | 全球边缘节点，低延迟，免运维 |
| 存储方案 | 无（纯内存） | 降低成本，简化架构，保护隐私 |
| 前端方案 | 内嵌 HTML | 单文件部署，无 CDN 依赖 |
| API 选择 | remove.bg | 成熟稳定，效果好，按量计费 |
| 文件传输 | FormData + ArrayBuffer | 浏览器原生支持，无需额外库 |

### 4.3 API 设计

#### POST /api/remove-bg

**请求：**
```
Content-Type: multipart/form-data
Body: image (File)
```

**成功响应（200）：**
```json
{
  "success": true,
  "image": "data:image/png;base64,..."
}
```

**错误响应：**
```json
{
  "error": "错误描述"
}
```

| 状态码 | 场景 |
|--------|------|
| 400 | 文件格式不支持 / 文件过大 / 未上传文件 |
| 402 | remove.bg 免费额度用完 |
| 403 | API Key 无效 |
| 500 | 服务器错误 / API Key 未配置 |

## 5. 非功能需求

| 维度 | 要求 |
|------|------|
| **性能** | 页面加载 < 1s，API 响应取决于 remove.bg（通常 5-15s） |
| **可用性** | 依赖 Cloudflare SLA（99.9%） |
| **限制** | 免费版 Worker：100KB 请求体；付费版：无限制 |
| **安全** | API Key 通过 Wrangler Secret 存储，不暴露在代码中 |
| **隐私** | 图片仅在内存中处理，不落盘，不记录 |

## 6. 部署与运维

### 6.1 部署步骤

```bash
# 1. 安装依赖
npm install

# 2. 登录 Cloudflare
npx wrangler login

# 3. 配置 API Key（安全存储）
npx wrangler secret put REMOVE_BG_API_KEY

# 4. 部署
npm run deploy
```

### 6.2 环境变量

| 变量名 | 说明 | 获取方式 |
|--------|------|----------|
| REMOVE_BG_API_KEY | remove.bg API 密钥 | [remove.bg/api](https://www.remove.bg/api) |

### 6.3 监控

- 查看日志：`npx wrangler tail`
- Cloudflare Dashboard 查看请求数、错误率、延迟

## 9. 验收标准

- [ ] 用户可以拖拽或点击上传 JPG/PNG/WebP 图片
- [ ] 超过 10MB 的图片被拦截并提示
- [ ] 上传后显示原图预览
- [ ] 点击处理后 15 秒内返回结果
- [ ] 结果图背景透明（棋盘格可验证）
- [ ] 可以下载 PNG 透明底图
- [ ] 可以重新上传新图片
- [ ] API 错误有友好提示
- [ ] 页面在手机端正常显示
