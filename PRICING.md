# 定价页设计文档

## 1. 定价结构

| 身份 | 条件 | 功能 |

| 方案 | 额度 | 价格(USD) | 单价 | 推荐 |
|------|------|-----------|------|------|

### 2.2 FAQ 详细内容

**Q: 积分包和订阅有什么区别？**
A: 积分包是一次性购买，积分永不过期，适合偶尔使用的用户。订阅是每月自动续费，适合高频使用的用户，单价更优惠。

**Q: 积分会过期吗？**
A: 不会。积分包购买后永不过期，随时使用。

**Q: 如何取消订阅？**
A: 随时在个人中心一键取消，当月有效期内继续使用，不会自动续费。

**Q: 图片会被保存吗？**
A: 不会。所有图片纯内存处理，处理完立即销毁，不存储任何数据。

**Q: 支持哪些付款方式？**
A: 目前支持 PayPal，后续将支持信用卡。

**Q: 有 API 接口吗？**
A: MVP 阶段暂未开放，后续会推出 API 方案，敬请期待。

**Q: 免费版有什么限制？**
A: 注册即送 3 次免费额度，之后每天可免费使用 3 次，最大支持 5MB 文件。

**Q: 可以退款吗？**
A: 支持 7 天无理由退款，请联系客服处理。

### 2.3 交互细节

1. Tab 切换：积分包 / 月订阅，默认显示积分包
2. 未登录用户点击购买 → 跳转 Google 登录 → 登录后回到定价页
3. 免费用户看到当前剩余积分
4. Pro 用户看到"当前套餐"标识
5. 移动端：卡片纵向排列
